# Rosemount

A native iOS community social networking app built on the ActivityPub protocol,
with full interoperability with Mastodon and Pixelfed.

## What is Rosemount?

Rosemount is designed for tight-knit communities — high school reunion groups, youth sports
leagues, religious congregations, neighborhood associations, and hobby clubs. It combines
familiar social features (photo sharing, direct messaging, location sharing, following people)
with the openness of the federated web.

## Key Features

- **Communities** — Create public or private groups; invite by handle, link, or QR code
- **Photo Sharing** — Multi-photo posts with editing, federated to Pixelfed and Mastodon
- **Direct Messages** — End-to-end encrypted 1-on-1 and group chats
- **Location Sharing** — Privacy-first opt-in live location and place sharing
- **ActivityPub Federation** — Follow and interact with users on any Mastodon or Pixelfed instance
- **Sign in with Mastodon / Pixelfed** — Use your existing federated identity
